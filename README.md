## Case studies

### 01 — Reducing Harassment by 60% on Divar
**Platform:** Divar · Iran's largest classified ads marketplace (40M+ users)  
**Problem:** Users making phone calls to strangers through the platform faced harassment, fraud, and privacy exposure.  
**What I did:** Led the UX for a Secure Call system with Trust & Safety — number masking, abuse reporting, behavioral signals.  
**Outcome:** 60% reduction in harassment incidents across 2.1M calls per week.

→ [Read the case study](https://mrsadri.github.io/Portfolio/case-studies/divar-secure-call.html)

---

### 02 — Turning One-Time Visitors into Lifelong Customers
**Platform:** Setare Aval Co. · Content & services platform  
**Problem:** High first-visit drop-off with no re-engagement loop in place.  
**What I did:** Cross-functional retention strategy — identified friction points through data, redesigned key flows, introduced engagement mechanics.  
**Outcome:** Measurable lift in return-visit rate and sustained engagement.

→ [Read the case study](https://mrsadri.github.io/Portfolio/case-studies/setare-aval-engagement.html)

---

### 03 — From 4% to 47%: Turning a Failing Feature Into Core Value
**Platform:** SetareYek App  
**Problem:** Bill payment existed in the app but almost nobody used it (4% adoption).  
**What I did:** User research to find why, then a ground-up redesign of the flow removing every friction point I found.  
**Outcome:** 47% adoption — a 10× increase. Bill payment became one of the app's top-used features.

→ [Read the case study](https://mrsadri.github.io/Portfolio/case-studies/setare-yek-bill-payment.html)

## About this repository

This is the source for [mrsadri.github.io/Portfolio](https://mrsadri.github.io/Portfolio) — built with **zero dependencies**: pure HTML, CSS, and JavaScript. No framework, no build step, no bundler.

The choice is intentional. A portfolio should load instantly and every decision in it should be explainable — the same standard I apply to the products I design.

**Quick start:**
```bash
# No install needed
open index.html
# or
python3 -m http.server 8000
```

**CSS architecture:** design tokens → base → layout → components → page-specific files. Dark mode by default, persisted via `localStorage`.
