Setare Aval (MCI) is one of Iran's leading telecommunications companies, providing services to millions of users across mobile, internet, and digital services. The platform needed to evolve from attracting users through promotions to building genuine long-term engagement and loyalty.

The Challenge

Setare Aval was experiencing high user acquisition through promotional campaigns, but users were churning quickly after the initial attraction period. The platform needed to shift from short-term promotional engagement to building sustainable, long-term user relationships.

Key challenges:

  • High user churn rate after promotional periods
  • Low long-term engagement metrics
  • Users not discovering value beyond initial offers
  • Need to build brand loyalty and retention

The goal was clear: Transform short-term users into long-term loyal customers.

Understanding the Problem

User Research

Through comprehensive user interviews and data analysis, we identified key insights:

User Pain Point #1
"I signed up for the promotional offer but didn't know what else the platform offered. After the promotion ended, I didn't see value in staying."
User Pain Point #2
"The onboarding process was too focused on the promotion. I never learned about the full range of services available."
User Pain Point #3
"I felt like the platform only cared about me during the promotion. After that, I was just another number."

Key Findings

  • Users lacked awareness of platform's full value proposition
  • Onboarding was too transactional, not relationship-building
  • No personalized discovery of features post-signup
  • Missing engagement triggers between promotional periods

The Solution

Strategic Approach

We redesigned the user journey to focus on relationship-building rather than transactional engagement:

1. Progressive Value Discovery

Implementation & Iteration

Phase 1: Onboarding Redesign

  • Multi-step onboarding that introduces 3-4 core features
  • Interactive tutorials with actual value demonstration
  • Progress tracking to build early momentum

Phase 2: Engagement Layer

  • Smart notification system with preference learning
  • Weekly digest emails highlighting user achievements
  • In-app celebration moments for milestones

Phase 3: Retention Features

  • Loyalty program with tiered benefits
  • Personalized dashboard showing user journey
  • Proactive support for users at risk of churning

What I Learned

1. Relationships Over Transactions

Building genuine relationships with users creates more value than one-time promotional interactions. Focus on long-term value delivery.

2. Progressive Discovery Works

Users don't need to see everything at once. A well-designed progressive disclosure system helps users discover value naturally over time.

3. Engagement Requires Context

Every interaction should add value. Contextual, value-driven notifications outperform promotional messages in building loyalty.

4. Celebrate Small Wins

Recognizing user milestones and achievements creates emotional connections that drive long-term engagement.
